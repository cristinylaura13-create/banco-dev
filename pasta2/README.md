# Banco Dev

Projeto desenvolvido em HTML e CSS para de estrutura,layout, responsividade e organização da páginas web.

## Tecnologias utilizadas
- HTML5
- CSS3

## Objetivo
Criar uma interface inspirada em aplicativo bancário, com foco em
organização visual e boa práticas básicas.

## Estrutura
.Cabeçalho com identificação do sistema

- Cabeçalho com identificação do sistema
- Bloco de saldo
- Botões de ação
- Lista de transações
- Card informativo
- Responsividade para celular
